Welcome to Fake Bank App

Manage your accounts and users easily along with making transactions

Structure of the app

Users Screen

Displays all users and CRUD functionalities

Delete user first deletes any transaction to/from that user and then deletes all accounts linked with that user and then finally deletes the user

Account Screen

Displays all accounts and CRUD functionalities

Delete Account: first deletes any transaction to/from that account and then deletes the account

Transaction Screen

Displays all transaction and CR functionalities

Transaction can not be directly deleted, it is only deleted when an account or a user is deleted.
